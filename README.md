# Babana Arline 🛵

Ever jumped on a bendskin in Douala or Yaoundé and wondered
"what if something goes wrong and nobody knows where I am?"

That's exactly why I built this.

Babana Arline is a mobile app that brings a basic layer of safety
to moto-taxi transport in Cameroon — real-time ride tracking,
driver verification, and an automatic alert system that detects
when something feels off about a trip.

This repo is the backend that powers it all.

## What it does
- Lets passengers share their live ride with a trusted contact
- Verifies drivers before they can accept rides
- Detects suspicious route deviations automatically
- Gives passengers a one-tap SOS button when things go wrong
- Handles phone number login (no passwords, no email required)

## Project Structure
- `accounts/` — User authentication and profiles
- `rides/` — Ride requests and tracking
- `alerts/` — Anomaly detection and SOS
- `ratings/` — Post-ride feedback
